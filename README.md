# poc-rabbitmq
Poc vagrant + rabbitmq

Inicialiar os ambientes com vagrant - node1 e node2
--------------------------------------------------------------
  cd poc-rabbitmq/
  vagrant up
  
  

Verificar o status das maquinas(node1/node2)
---------------------------------------------------------------
  vagrant status
  
node1                     running (virtualbox)
node2                     running (virtualbox)


Acesso as maquinas
---------------------------------------------------------------
  vagrant ssh node1
  vagrant ssh node2
